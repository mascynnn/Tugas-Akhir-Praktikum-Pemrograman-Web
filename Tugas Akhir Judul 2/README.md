## âš™ï¸ Langkah Instalasi dan Penggunaan

1. **Pastikan Git sudah terinstal di komputer**
Buka Command Prompt di device anda lalu,

   * Cek dengan perintah:

     ```bash
     git --version
     ```
   * Kalau belum terinstal, unduh dan install Git dari [git-scm.com](https://git-scm.com/).

2. **Buka Command Prompt / Terminal**

   * Windows: tekan `Win + R`, ketik `cmd`, tekan Enter.
   * Mac/Linux: buka Terminal.

3. **Pilih lokasi penyimpanan proyek**

   * Gunakan perintah `cd` untuk pindah ke folder tujuan. Contoh:

     ```bash
     cd D:\Kuliah\Kuliah Semester 5\Praktikum Pemrograman Web
     ```

4. **Clone repositori dari GitHub**

   ```bash
   git clone https://github.com/mascynnn/Tugas-Akhir-Praktikum-Pemrograman-Web.git
   ```

   * Perintah ini akan menyalin semua file proyek ke komputer lokal.

5. **Masuk ke folder proyek**

   ```bash
   cd "Tugas Akhir Judul 2"
   ```

6. **Buka file `index.html` di browser**

   * Klik dua kali file `index.html` atau buka lewat browser pilihan kamu (Chrome, Edge, Firefox, dll.).
   * Website portofolio akan tampil persis seperti desain aslinya.

7. **Tidak diperlukan instalasi tambahan**

   * Karena website ini sepenuhnya berbasis HTML & CSS murni, semua fitur sudah bisa digunakan langsung.
